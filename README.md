# 🛒 E-Commerce Website

A responsive and interactive E-Commerce Website built using **HTML, CSS, and JavaScript**.  
This project showcases a modern shopping interface with product listings, cart functionality, and a user-friendly design.

---

## 🚀 Features

- 🛍️ Product Listing Page
- ➕ Add to Cart Functionality
- 🧮 Dynamic Cart Price Calculation
- 📱 Responsive Design
- 🎨 Clean and Modern UI
- 🔍 Product Search & Navigation
- ⚡ Interactive User Experience

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript**

---

## 📂 Project Structure

```bash
E-commerce-Website/
│── index.html
│── style.css
│── script.js
│── images/
```

---

## 💻 How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/your-username/E-commerce-Website.git
```

2. Open the project folder

```bash
cd E-commerce-Website
```

3. Run `index.html` in your browser
